## 🎨 How to Use (Web Interface)

### Even Simpler! Everything in Browser Control Page:

```
1. Browser Control page → Initialize Browser (one time)

2. For each action (all in same page!):
   └─> Enter prompt in "Test Prompt" field: "click login button"
   └─> Click "Execute in Browser"
   └─> System auto-generates code AND executes it
   └─> Browser performs action

3. Browser Control page → Close Browser (when done)
```

**No page switching! No copy/paste! Just enter prompts and execute!**
