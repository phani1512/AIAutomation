## Usage Guidelines

### How to Use This Reference

1. **Find the Right Prompt:**
   - Browse by dataset category (Common Web Actions, Selenium Methods, Element Locators, Sircon UI)
   - Use alphabetical listing for Sircon UI prompts
   - Check pattern categories for generic patterns

2. **Use Prompts in Your Code:**
   - Copy exact prompt text
   - Replace placeholders like `{EMAIL}`, `{PASSWORD}`, `{URL}` with actual values
   - Combine prompts using "and" for multi-step workflows

3. **Examples:**
   ```
   Single prompt:
   "click search button"
   
   With placeholder:
   "enter john.doe@example.com in email field"
   
   Multi-step workflow:
   "navigate to https://example.com and enter John in first name field and click submit button"
   ```

4. **Best Practices:**
   - Use exact prompt text (case-sensitive)
   - For generic actions, check Common Web Actions first
   - For Selenium API calls, reference Selenium Methods
   - For application-specific Sircon actions, use Sircon UI prompts
