📖 Complete Workflow Example

This example shows testing a complete login-to-purchase flow on https://www.saucedemo.com

1
Initialize Browser
Open the Browser Control page and initialize the browser. It will stay open for all subsequent steps.
In Browser Control Page:
Browser Type:
Headless: (Uncheck to see browser)
✅ Browser opens and stays ready for commands
2
Navigate to Website
Switch to Generate Code page and use an AI prompt to navigate
📝 AI Prompt: "navigate to https://www.saucedemo.com"
driver.get("https://www.saucedemo.com")
Copy the code → Go to Browser Control → Paste → Execute
✅ Page loads in the browser
3
Login to Application
📝 AI Prompt: "enter standard_user in username field and secret_sauce in password field then click login button"
driver.find_element(By.ID, "user-name").send_keys("standard_user") driver.find_element(By.ID, "password").send_keys("secret_sauce") driver.find_element(By.ID, "login-button").click()
✅ User logged in successfully
4
Sort Products
📝 AI Prompt: "select Price (low to high) from product sort dropdown"
from selenium.webdriver.support.ui import Select dropdown = Select(driver.find_element(By.CLASS_NAME, "product_sort_container")) dropdown.select_by_visible_text("Price (low to high)")
✅ Products sorted by price
5
Add Product to Cart
📝 AI Prompt: "click add to cart button for the first product"
driver.find_element(By.XPATH, "//button[contains(text(), 'Add to cart')]").click()
✅ Product added to cart
6
Verify Cart Badge
📝 AI Prompt: "verify shopping cart badge shows 1"
cart_badge = driver.find_element(By.CLASS_NAME, "shopping_cart_badge") assert cart_badge.text == "1", f"Expected 1, got {cart_badge.text}"
✅ Cart badge verified
7-11
Continue Workflow...
Continue with: Go to Cart → Checkout → Fill Form → Complete Order
⬇️
All in the same browser session! No restarts needed.
✓
Close Browser
When workflow is complete, close the browser
